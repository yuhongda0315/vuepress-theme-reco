#### 功能描述
当群组名称变更时，App Server 应该调用此接口刷新在融云侧保存的群组信息，以保证融云发送远程推送（PUSH）提醒的时候，能够正确显示相关内容信息。

刷新群组信息后 5 分钟内生效，Push 推送时将显示刷新后的群组名称。

<span style="color:red;">时序图</span>

调用频率：无限制

|名称|必填|长度限制|限制说明|
| :---------:| :---------:|:---------:| :-------|
|群组 Id|必传|30 个字符|要修改的群组的 Id|
|群组名称|必传|60 个字符|要修改的群组的名称|
